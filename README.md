# MEVN boilerplate

based on [this tutorial](https://medium.com/@anaida07/mevn-stack-application-part-1-3a27b61dcae0)


## prerequisites
- mongodb [installation instructions](https://docs.mongodb.com/manual/administration/install-community/)

## running
api:
```
$ cd api/
$ npm install
$ npm run start
```
web interface:
```
$ cd web/
$ npm install
$ npm run dev
```
